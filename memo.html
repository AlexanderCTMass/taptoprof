Executive summary

TapToProf: AI-нативная платформа для blue‑collar

Интегральный бизнес-план и аналитический отчёт. Март 2026, Pre‑Seed.

Концепция: TapToProf — глобальная AI-платформа для поиска специалистов (blue‑collar), заменяющая «человека в процессе» AI-агентами и токсичную модель комиссий (20‑30%) на прозрачную SaaS-подписку для мастеров ($29.99/мес).

Проблема: Рынок бытовых услуг США оценивается в $543 млрд (2025‑26). Лидеры (Angi, TaskRabbit) продают «мусорные» лиды по $15–120 или взимают двустороннюю комиссию до 30% (BBB, 2024). Мастера теряют 80% времени на переписку, клиенты тонут в спаме.

Решение: Zero‑UI (фото+голос) → AI-оркестратор → прозрачный аукцион + подписка для мастеров. Консервативный LTV мастера — $720, CAC ~$180, LTV/CAC = 4. При 10 000 мастеров ARR достигнет $3.6 млн (только подписка).

legacy is broken

Проблема: болевые точки рынка США

Со стороны клиента

Высокий порог ввода: чтобы вызвать сантехника, нужно заполнить анкету, сравнить 10+ откликов. Отсутствие доверия: фото работ могут быть скачаны из Pinterest, верификация — главный сдерживающий фактор (Juniper Research, 2024).

Со стороны мастера

Angi продаёт один лид 3‑8 подрядчикам, цена закрытой сделки достигает $1400–2500 (BBB жалобы, 2025). TaskRabbit забирает 15‑30% комиссии + сбор с клиента 5‑9% (TaskRabbit ToS, 2026). Мастер с заказа $100 получает на руки ~$80.

Структурный застой

Монополистам выгоден хаос: они монетизируют каждый контакт. Рынок созрел для disruption.

Service‑as‑a‑Software

Решение: SaaS вместо комиссий

AI‑операционная система для услуг. Ключевые инновации: Zero‑UI, мультиагентная система, прозрачный аукцион, синтезированная репутация, подписка $29.99/мес.

user journeys

Пользовательские сценарии (США)

Клиент Анна (Денвер): фотографирует кран, говорит голосом → AI распознаёт, запускает аукцион среди ближайших сантехников. Через минуту получает предложение с ценой $90 и подтверждением.

Мастер Александр: открывает приложение, видит заявки, ставит цену или включает «автопилот» (AI торгуется в пределах $80‑120). В конце месяца платит $29.99 подписки.

multi‑agent orchestration

AI‑мультиагентная система

Оркестр агентов: Classifier (CV+NLP), Scheduler (календари, Maps), Negotiator (прогноз цены, автоторги), Reputation (парсинг Google Maps, Yelp, выявление накруток), Summary (отзывы, отчёты).

Стек: Python (FastAPI), LangChain, OpenAI/Claude, YOLOv8, PostgreSQL (pgvector), Stripe Connect.

SWOT

Силы: подрывная цена ($30/мес vs 30% комиссии), AI‑удобство, прозрачность.
Слабости: проблема «курица‑яйцо», верификация, комиссии эквайринга.
Возможности: миграция мастеров от Angi, B2B (property managers), партнёрства с Home Depot.
Угрозы: ответ Google LSA, регуляторика (AB5 в Калифорнии).
