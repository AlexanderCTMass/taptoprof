Executive summary

TapToProf: AI-нативная платформа для blue‑collar

Интегральный бизнес-план и аналитический отчёт. Март 2026, Pre‑Seed.

Концепция: TapToProf — глобальная AI-платформа для поиска специалистов (blue‑collar), заменяющая «человека в процессе» AI-агентами и токсичную модель комиссий (20‑30%) на прозрачную SaaS-подписку для мастеров ($29.99/мес).

Проблема: Рынок бытовых услуг США оценивается в $543 млрд (2025‑26). Лидеры (Angi, TaskRabbit) продают «мусорные» лиды по $15–120 или взимают двустороннюю комиссию до 30% (BBB, 2024). Мастера теряют 80% времени на переписку, клиенты тонут в спаме.

Решение: Zero‑UI (фото+голос) → AI-оркестратор → прозрачный аукцион + подписка для мастеров. Консервативный LTV мастера — $720, CAC ~$180, LTV/CAC = 4. При 10 000 мастеров ARR достигнет $3.6 млн (только подписка).

legacy is broken

Проблема: Анализ болевых точек рынка США

Проблемы клиента (Спрос)

Высокий порог ввода: чтобы вызвать сантехника, нужно заполнить анкету, сравнить 10+ откликов. Отсутствие доверия: фото работ могут быть скачаны из Pinterest, верификация — главный сдерживающий фактор (Juniper Research, 2024). Усталость от выбора и спам: получить 10-15 откликов — не подарок, а наказание.

Проблемы мастера (Предложение)

Angi продаёт один лид 3‑8 подрядчикам, цена закрытой сделки достигает $1400–2500 (BBB жалобы, 2025). TaskRabbit забирает 15‑30% комиссии + сбор с клиента 5‑9% (TaskRabbit ToS, 2026). Мастер с заказа $100 получает на руки ~$80. 80% времени мастера тратится на «офисную работу»: переписки, звонки, уточнения.

Структурный застой рынка

Монополистам выгоден хаос: они монетизируют каждый контакт. Рынок созрел для disruption.

Service‑as‑a‑Software

Решение: Service‑as‑a‑Software (SaaS) — Новая парадигма

AI‑операционная система для услуг. Ключевые инновации: Zero‑UI, мультиагентная система, прозрачный аукцион, синтезированная репутация, подписка $29.99/мес.

user journeys

Пользовательские сценарии (США)

Сценарий для клиента (Анна, Денвер)

Инпут: Анна открывает приложение, фотографирует кран, говорит: "Вот тут течет, под раковиной. Нужно починить. Я дома после 7 вечера". AI‑обработка: CV определяет тип смесителя, NLP извлекает суть. Аукцион: Система отправляет заявку ближайшим сантехникам. Торги: Один ставит $90, другой $85. Выбор: AI предлагает Анне: "Найден сантехник Александр. Рейтинг 4.9. Цена $90. Подтвердить?" Результат: Анна нажимает "Да", мастер получает адрес.

Сценарий для мастера (Александр, сантехник)

Онбординг: Скачивает приложение, даёт доступ к геолокации, AI парсит его профиль (Google Maps, Yelp). Работа: Видит ленту заявок, нажимает "Участвовать". Автопилот: Включает режим, AI торгуется в пределах $80‑120. Оплата: В конце месяца платит $29.99 подписки. Никаких комиссий.

multi‑agent orchestration

Технология: Архитектура AI‑мультиагентной системы

Оркестр агентов: Agent‑Classifier (CV+NLP) определяет задачу; Agent‑Scheduler (календари, Maps) согласует время; Agent‑Negotiator анализирует ставки, прогнозирует цену, ведет торги; Agent‑Reputation парсит открытые данные (Google Maps, Yelp, соцсети), выявляет накрутки; Agent‑Summary собирает отзыв, генерирует чек.

Стек: Python (FastAPI), LangChain / AutoGen, OpenAI/Claude/Llama 3, YOLOv8/CLIP, PostgreSQL (pgvector), Stripe Connect.

Анализ рынка и конкурентная среда в США

Оценка рынка (TAM, SAM, SOM)

TAM (США, бытовые услуги) = $543 млрд (IBISWorld, 2026). Другие оценки — до $870 млрд.

SAM (онлайн‑сегмент) — десятки миллиардов долларов, точные данные отсутствуют, требуется уточнение.

SOM: ниша AI‑подписки, первые 500 мастеров в пилотном городе.

Детальный разбор ключевых конкурентов

Angi: тысячи жалоб в BBB, автоматическое продление, комиссии за расторжение до $880. TaskRabbit: высокий процент, мастер получает $80‑85 со $100. Структурная проблема: платформы не могут предложить локализованных решений и гибких моделей.

SWOT-анализ для выхода на рынок США

Силы: подрывная цена ($30/мес vs 30% комиссии), AI‑удобство, прозрачность аукциона.
Слабости: проблема «курица‑яйцо», верификация, комиссии эквайринга.
Возможности: миграция мастеров от Angi (из-за скандалов), B2B (property managers), партнёрства с Home Depot/Lowe's.
Угрозы: ответ Google LSA, регуляторика (AB5 в Калифорнии).
